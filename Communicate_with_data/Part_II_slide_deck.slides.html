Part II - Prosper Loans Dataset¶

by Franck Loïc NOUMOU TCHOMEGNI¶

Investigation Overview¶

Dans cette analyse, je voulais comprendre deux choses :

  • Quels sont les facteurs qui influencent la note Prosper ?
  • Les facteurs qui peuvent être utilisés pour prédire le défaut de crédit.

Dataset Overview¶

Cet ensemble de données contient 113 937 prêts avec 81 variables sur chaque prêt. Aux fins de cette enquête, j'ai pris les variables suivantes : Term, LoanStatus, BorrowerAPR, BorrowerRate, ProsperRating (Alpha), ProsperScore, ListingCategory (numeric), EmploymentStatus, DelinquenciesLast7Years, StatedMonthlyIncome, TotalProsperLoans , LoanOriginalAmount, LoanOriginationDate, Recommendations, Investors. Après ajustement, l'ensemble de données contenait 25 938 lignes de données.

Il y avait trois problèmes majeurs avec les données :

  • Prosper a commencé à utiliser la notation Prosper à partir de 2009. Avant 2009, Prosper utilisait la note de crédit. J'ai dû supprimer entièrement les points de données antérieurs à 2009

  • La plupart des prêts dans l'ensemble de données sont en fait des prêts en cours et ne peuvent pas être utilisés pour déterminer comment le défaut de crédit peut être prédit. Les crédits en défaut ont été divisés en deux catégories imputés et en défaut (voir sous-diapositive). Pour résoudre ce problème, j'ai supprimé tous les prêts actuels et fusionné les crédits imputés et en défaut.

  • Le revenu mensuel déclaré avait un nombre inhabituellement élevé de valeurs aberrantes et une très large plage de valeurs allant de 0 à 1 750 000. Pour résoudre ce problème, j'ai supprimé toutes les données au-dessus de trois écarts-types par rapport à la moyenne.

ProsperRating (Apha) est une variable catégorielle ordinale avec les valeurs suivantes du plus élevé au plus bas ('AA', 'A', 'B', 'C', 'D', 'E', 'HR'), les notes ont une distribution presque normale dans ensemble de données centré sur 'C', les cotes 'AA' les plus élevées et les cotes 'HR' les plus basses ont le moins d'occurrences.
